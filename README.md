
Технології
- **Frontend:** React, TypeScript, WebSocket, Axios
- **Backend:** NestJS, WebSocket, REST API
- **База даних:** PostgreSQL (TypeORM)
- **Інші:** Vite, ESLint, Prettier


Швидкий старт


git clone https://github.com/Kapysta017/Mafia.git


cd Mafia

Залежності та запуск:


Клієнт:
- cd client
- npm install
- npm run dev


Сервер:
- cd server
- npm install
- npm run start:dev
